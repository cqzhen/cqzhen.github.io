# Linux 基础篇

### 系统篇

    uname -a 查看系统信息
		source /etc/profile 刷新权限
### 磁盘篇

    df -hl 查看磁盘剩余空间
    
    df -h 查看每个根路径的分区大小
    
    du -sh [目录名] 返回该目录的大小
    
    du -sm [文件夹] 返回该文件夹总M数
    
    du -h [目录名] 查看指定文件夹下的所有文件大小（包含子文件夹）
